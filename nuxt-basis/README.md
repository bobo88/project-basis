# nuxt-web
#### *本项目基于nuxt搭建，主要是做PC + M端（移动端）适配 ，服务端渲染SSR，响应式官网开发*
### 一、项目运行
+ > *npm install*  ( 可简写：*npm i* )
+ > *npm run dev*
+ > <u>Node高版本会有错误提示，请用nvm工具将node版本切换在v14.18.0左右</u>
