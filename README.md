# project-basis
#### 项目基础框架： 包含 webpack/Vue/React/小程序/NodeJS/Koa 等  

--------------------------------------
+ ⭐ **webpack-basis:**  
    webpack基础框架搭建  

+ ⭐ **vue-3-basis:**  
    vue3.x框架搭建，结合TS  

+ ⭐ **react-basis:**  
    react基础框架搭建，搭配目前最新版本 + hooks  

+ ⭐ **nodejs-basis:**  
    探索Nodejs的微服务以及高级特性，大前端范畴  

+ ⭐ **gulp-basis:**  
    重温gulp的流配置  

+ ⭐ **applets-basis:**  
    小程序相关的框架搭建，引申 uni-app 等  

+ ⭐ **koa-basis:**  
    koa框架的基础实现  
    
+ ⭐ **nuxt-basis:**   
    探索vue的SSR技术，引申 next.js(React对应框架)
